Permite agregar tags de Bitacoras.com en nuestros post.

== Description ==

Con este plugin podr&aacute;s generar r&aacute;pidamente etiquetas para que tus post puedan ser indexados por Bitacoras.com
sin necesidad de pegar el c&oacute;digo HTML necesario para cada una de tus anotaciones 

&iquest;C&oacute;mo integrar a tu blog? S&oacute;lo inserta este pedazo de c&oacute;digo en tu plantilla donde quieres que sea visualizado 
los tags `<?php the_bita_tags(); ?>` 

PS: M&aacute;s instrucciones [Bitatags](http://diavolo.us/bitatags/) .

== Installation ==

Sube el plugin Bitatags a tu blog, activa y listo.
